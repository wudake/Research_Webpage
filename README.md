# 我的个人网站 (Research_Webpage)

这是一个简单的静态网站，托管在 GitHub Pages 上。

## 访问地址
https://wudake.github.io/Research_Webpage/

## 如何更新网站

1. 修改 `index.html`, `about.html`, 或 `contact.html` 文件。
2. 在终端运行以下命令提交更改：
   ```bash
   git add .
   git commit -m "更新网站内容"
   git push
   ```
3. 等待几分钟，GitHub Pages 会自动更新。

## 本地预览

直接双击打开 `index.html` 文件即可在浏览器中预览。
